Ecael Bot Privacy Policy 
Effective Date: February 21, 2026

1. Data Collection
Ecael Bot only collects publicly available server information to provide real-time status updates. No personal Discord data (messages, user IDs, or DMs) is stored or shared.

2. Use of Data
Collected server data is used solely to display server status, player counts, and map information in Discord.

3. Data Sharing
Ecael Bot does not sell, trade, or share any personal data with third parties.

4. Data Retention
Server status data may be temporarily cached to improve performance, but no personal data is retained.

5. User Rights
You can stop using Ecael Bot at any time. No personal information is stored, so no deletion requests are necessary.

6. Security
AZR implements standard measures to protect server data and prevent unauthorized access.

7. Contact
For questions about this Privacy Policy, contact the developer: AZR
